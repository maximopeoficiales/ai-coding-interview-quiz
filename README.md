# AI Coding Interview Quiz

Quiz responsive para practicar conceptos de entrevista tecnica para un rol de AI Coding Engineer.

## Incluye

- 70 preguntas de opcion multiple (3 opciones, 1 correcta)
- Modo por bloques (7 bloques de 10 preguntas)
- Modo examen completo (70 preguntas mezcladas)
- Feedback inmediato con explicacion
- Resultados finales con desglose por bloque
- Opcion de repetir solo preguntas falladas

## Como usar

1. Abre el sitio publicado en GitHub Pages.
2. Elige Examen Completo o un bloque especifico.
3. Responde cada pregunta y revisa la explicacion.
4. Al final, revisa tus errores y repite falladas.
